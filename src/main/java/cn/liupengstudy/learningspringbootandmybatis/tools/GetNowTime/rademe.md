# 获取时间类工具

## 原理

> 通过Date对象获取各种时间数据

## 构造过程

* 构建抽象类，内部创建一个Date对象

* 各种功能单一的类集成抽象类，拓展出方法

## 获取当前时间

### 用法

`new GetNowTimeStamp().getInformation();`


## 获取当前日期

### 用法

`new GetNowDate().getInformation();`

## 获取当前日期-时间

### 用法

`new GetNowDateTime().getInformation();`

## 获取当前时间戳

### 用法

`new GetNowTime().getInformation();`
